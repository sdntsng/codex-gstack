# Plan CEO Review

Use this skill to challenge a plan at the product and company level.

## Workflow
1. Read the current plan or request.
2. Restate the user problem in plain language.
3. Ask whether the plan is solving the right problem or merely the literal request.
4. Evaluate four possible moves:
   - expand the ambition
   - selectively expand only the highest-leverage parts
   - hold scope and sharpen execution
   - reduce scope to the most defensible wedge
5. Recommend one direction and explain why.
6. Rewrite the plan so the recommendation is actionable.

## What this skill should improve
- user empathy
- product sharpness
- premium or differentiated experience
- wedge selection
- scope discipline

## Output expectations
Return:
- current plan summary
- what problem the plan is really solving
- recommended direction
- revised product framing
- explicit tradeoffs

## Guardrails
- Do not confuse bigger scope with better product.
- Push for a clear wedge, not a vague platform story.
- Preserve realism: ambition is good, incoherence is not.
